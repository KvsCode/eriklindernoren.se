Geolocation Application

iOS development

This was a project I did to get more practice with Swift. Bra Bärs

iOS development

'Bra Bärs', read in English as "Good beer", was a idea me and my classmates came up with one evening while we were sitting at our favourite pub in Linköping. The idea of the app was to be able to navigate the beers available at the Swedish alcohol monopoly Systembolaget and sort beers by professional and customer reviews, by prices and by the users favourite type of beer.
Similar products are available for wines, but as there was nothing like that for beers and the fact that we ourself felt that it was something that we would use made us go ahead and start building.
I used the API supplied by Systembolaget to get information about the current supply, and added images from a separate API. There was no way of requesting data from Systembolaget specified by categories at a time, and I thereby decided to build a XML-parser in Java to seperate the data that was relevant to that which was not.

Programming languages and tools used:
  • Swift
  • Java
APIs used:
  • Systembolaget.se API
  • Systemetbevakningsenheten.se API
  • Systembolagetapi.se
