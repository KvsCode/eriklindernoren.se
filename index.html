About

My name is Erik Linder-Norén and I'm currently studying towards a Master of Science in Computer Science and I'm really enjoying it this far. This site is meant to function as a online resume of mine but has also been my way of learning web development and web design.

I've always loved sketching, tinkering with things and exploring new tools to create with. The joy in building might have originated in my Lego days as a kid, and later evolved to putting computers together in my younger teen years to as of now really enjoying building with ones and zeros.

I love how programming enables people to quickly explore new ideas and I've found that the best, and most fun way for me to learn is to explore the languages and tech in small side projects parallel to my studies and current employment. The results of some of my side projects has spanned from iOS applications to goofy music production software for the OSX platform.

I have since june 2014 been employed by Ericsson as a software developer. I'm currently developing a command-line interface for handling communications with Radio Base Stations using the NETCONF protocol.
I'm a huge fan of all things basketball and watches just about every Boston Celtics game.

Some personal projects

Geolocation Application

 iOS development

This was a project I did to get more practice with Swift. It started out in me wanting to build a user interface for the Google Maps SDK and then evolved to adding as much location based information from APIs out there as I could find. The application could with the help of APIs from several photo sharing services display the most recently taken photo from the tagged location. The most interesting part in working on this project was using a bunch of APIs together and realizing how much great accessible data that's out there.

Programming languages and tools used

  • Google Maps SDK
  • OpenWeatherMap API
  • TimezoneDB
  • Flickr API
  • GeoLocation.ws API
  • Wikipedia API

Bra Bärs

 iOS development

The idea of the app is to be able to navigate the beers available at the Swedish alcohol monopoly Systembolaget and sort beers by professional and customer reviews, by prices and by the users favourite type of beer.
I used the API supplied by Systembolaget to get information about the current supply, and added images from a separate API.
There was no way of requesting data from Systembolaget specified by categories at a time, and I thereby decided to build a XML-parser in Java to seperate the data that was relevant to that which was not.

Programming languages and tools used
  • Swift
  • Java
  • Systembolaget.se API
  • Systemetbevakningsenheten.se API
  • Systembolagetapi.se API

Sonus

 Music composing app built using JFugue API

This was a project that I did with a classmate in a Java class that we had. The project was not meant to take up as much time that it did in the end, but it was really fun working on.
I was after the course ended employed by Ericsson for the summer and ended up building my project there solely in Java and I think that the time that we put into our Java project in the class before helped me out at Ericsson in the beginning.
Most other project groups ended up doing games as their project but we decided early on that we wanted to build a tool of some sort. We did a little research about fun modules out there for Java and we found JFugue.
JFugue makes it easy to compose music just by entering strings of the achords to be played and by what instrument.

Programming languages and tools used

  • JFugue API

This website

 Web development

I have wanted to get into learning web development for a long time and it was fun to finally find the inspiration. The idea of building sort of like a digital resume came from a friend who talked to me about building one himself and I thought it sounded like a fun way to get my web development ball rolling.
This site has been built in HTML, CSS and Javascript using Gulp as the build system. During the process of building it I have tried tools such as jQuery, Bootstrap, Animate.css, Wow.js, Font Awesome and a couple of Google Fonts. I have since getting a bit more accustomed to web development tried to limit my use of frameworks and plugins for structure, animations and such. As it stands right now everything on this site is custom built using only HTML, CSS, Javascript and JQuery.
Working on this site has been a great learning experience for me and I already look forward to the next web dev project.
